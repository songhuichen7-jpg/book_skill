# anthropic 风格 · 扉页插画 art-direction 前缀

生图时把下面这段作为「风格前缀」，再拼上「背景必须是纯色 <该章 hex>（<色名>）填满整张画布」+「画面内容：<该章 illustrationConcept>」。生图能力用你所在环境可用的图像生成命令行工具（如 `codex` 或任意可用的图像生成 CLI）来调用。

`$SKILL/scripts/gen_illustrations.py` 会自动读取本文件第一个代码块作为 STYLE 前缀。

```
为书的章节扉页绘制一幅插画并保存为 png。严格使用这种统一的编辑插画语言：白色的几何形状（三角形/正方形/圆形/八边形等）+ 粗黑色线条连接 + 实心黑色圆点作为端点/节点 + 少量简洁的手绘黑色细线元素。整体 playful、黑白高对比、极简、有手工感、大量留白、横向构图、主体偏画面一侧、另一侧大面积留白。除背景那一种颜色外，只允许白色与黑色；不要任何文字、不要渐变、不要 3D、不要纹理颗粒、不要写实。
```

## 每章 concept 的取材原则

呼应该章主题、用上面这套"白形状+黑线+圆点"语言能表达的具体意象。例：

- 循环/loop → 黑线绕成闭环，环上等距白圆点，中心一个白方块伸手绘线连外围小形
- 工具/接口 → 一凸榫一凹槽两个白形被黑线引导咬合，接缝处冒小圆点
- 记忆/存储 → 几块白色堆叠砖块，黑线双向取/存箭头，一张卡片被抽进上方循环
- 多 agent → 一个白色中心节点黑线辐射到 3–5 个白子节点（星型），旁边两对等节点一条更粗黑线
- 安全/护栏 → 白三角三顶点，一条边被手绘剪断符号切开，外围白虚线沙箱框
- 解剖/拆解 → 一个白色装置被黑线"爆炸图"拆成几个零件块，引线标注

每章 concept 由 TOC 阶段产出（写进 `$PROJECT/assets/illustration-specs.json` 的 `concept` 字段）。
