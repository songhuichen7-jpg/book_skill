# 风格：anthropic

对标 Anthropic 风格出版物的编辑设计语言。克制、人文、暖色纸感。

## 视觉令牌

| 维度 | 取值 |
|---|---|
| 开本 | 横向 Letter（11 × 8.5 in） |
| 内文底色 | **纯白 #FFFFFF**（关键：不要奶油/米黄，那会显土） |
| 主强调色 | clay 赤陶橙 `#D97757`（Anthropic 官方色）——**只用于封面 + 唯一强调色** |
| 章节扉页色 | 6 色循环：鼠尾草绿 `#609A81`、长春花紫 `#827DBD`、赭黄 `#C2913B`、灰蓝 `#5E7E96`、砖玫红 `#C2766A`、橄榄绿 `#87894F`（相邻章不撞色；clay 不用于章节） |
| 文字 | 近黑 `#191919`；次要灰 `#6B6B6B` |
| 标题字体 | 衬线：Hoefler Text + 宋体 |
| 正文字体 | 无衬线：Helvetica Neue + PingFang SC |
| 代码字体 | Menlo + PingFang SC |

这些字体是 macOS 自带。换机器先 `typst fonts | grep -iE "hoefler|songti|helvetica|pingfang|menlo"` 确认存在，缺了就在 `$SKILL/styles/anthropic/template.typ` 顶部换等价字体。

## 页面结构

- **封面**：整版 clay，衬线大标题左上，左下角手绘星芒标记 + 系列名。
- **每章扉页**：整版该章色 + 右上角插画 + 左下 `Chapter N` 药丸 + 衬线大标题。
- **目录页**：纯白，衬线"目录"，自动从隐藏的一级标题抓「标题 … 页码」。
- **内文页**：纯白，双栏正文（`prose`），代码/流程图/表/callout 跨整页宽。

## 插画语言（关键，决定整书质感）

- **白色几何形状（三角/方/圆/八边形）+ 粗黑线条连接 + 实心黑圆点 + 少量手绘黑色细线**，压在该章纯色底上。
- playful、黑白高对比、极简、大量留白、横向构图、主体偏一侧。
- **每章一张、各不相同，但呼应该章主题**（例如某章讲"连接"=接口咬合、某章讲"存储"=堆叠形态、某章讲"协作"=多节点群落…，按目标书的实际章节主题发挥）。
- 见 `$SKILL/styles/anthropic/illustration-prompt.md`（生图用的 art-direction 前缀）；生图用任意可用的图像生成 CLI/能力（如 `codex` 或其他生图工具），把该前缀拼到每章的主题描述前。

## 排版口味（写正文时遵循）

- 通俗易懂：概念用大白话 + 类比；代码硬核可运行。
- 每章：开头点出本章【核心问题】；至少 1 个 `keypoint`；结尾 1 个 `jd-box`（或要点框）；合适处用 `flow`/`table`/`note`/代码。
- 诚实讲权衡，直说不附和。

## 这个风格"为什么好看"（换风格时别丢的东西）

1. **页面底色 = 插画背景色** → 插画抠透明后无缝融入，没有贴图矩形。
2. **只一个强调色** → 不打架；扉页才上彩。
3. **衬线标题 + 无衬线正文** → 编辑感，不像 SaaS 落地页。
4. **细线 / 小 marker / 浅底** 的克制 callout 与流程图 → 不像 PPT。
